# 基于哈夫曼编码、香农编码和Shannon-Fano-Elias编码的信息论压缩软件

## 项目简介

本项目实现了一个基于信息论信源编码理论的文本压缩软件，支持多种编码方法（哈夫曼编码、香农编码、Shannon-Fano-Elias编码），并通过实验验证信源编码定理和熵极限。

## 项目结构

```
.
├── README.md                 # 项目说明
├── src/                      # 源代码目录
│   ├── encoder.py           # 编码器基类和实现
│   └── utils.py             # 工具函数
├── experiments/              # 实验脚本
│   ├── exp1_artificial.py   # 实验1：人工信源
│   ├── exp2_real_text.py    # 实验2：真实文本
│   └── exp3_markov.py       # 实验3：马尔可夫信源
├── data/                     # 测试数据
│   └── samples/             # 示例文本文件
├── results/                  # 实验结果
   └── figures/             # 生成的图表
```

## 使用方法

启动GUI界面：

```bash
python gui.py
```

GUI界面功能：

- 📁 文件选择：点击"选择文件"按钮上传要压缩/解压的文件
- 🔧 编码方式选择：支持哈夫曼、香农、Shannon-Fano-Elias三种编码
- 📊 实时显示：压缩统计信息（熵、平均码长、压缩率等）
- ⏳ 进度显示：压缩/解压过程实时进度条
- 💾 自动保存：自动生成输出文件路径


## 功能特性

### 核心功能

- ✅ 支持哈夫曼编码（最优前缀码）
- ✅ 支持香农编码
- ✅ 支持Shannon-Fano-Elias编码
- ✅ 计算信源熵和平均码长
- ✅ 统计压缩率和性能指标
- ✅ 完整的编码/解码功能

## 理论验证

本项目验证以下信息论核心结论：

1. **信源编码定理**：平均码长下界为熵 H(X)，上界可接近 H(X)+1
2. **哈夫曼码最优性**：在平均码长意义下，哈夫曼码是最优前缀码
3. **熵率与相关性**：利用符号相关性可以降低平均码长，接近熵率
